### 🚀 Sobre mim

Fundador da **RhoneyInc**, um studio de produtos digitais em Belém (PA) construído sob o conceito **"Uma conta. Todos os softwares."** — uma plataforma unificada para um portfólio de SaaS B2C e B2B.

---

### 🧱 Portfólio

| Produto | Descrição | Stack |
|---|---|---|
| **MeuPet** | Rede social para pets: feed, adoção, marketplace, geolocalização | HTML/JS · Supabase · TWA |
| **FinWise** | Educação financeira e conversor de moedas em tempo real | Lovable · Supabase |
| **FitNow** | App fitness com ligas, mentorias e integrações nutricionais | Lovable · Supabase · n8n |
| **MontaMovel** | SaaS B2B de gestão de montagem de móveis, 12 módulos | React · Supabase · Claude API |
| **Até Passar** | EdTech para concursos públicos | React · Supabase · Tailwind |
| **API de Futebol** | API pública com auth e rate limiting | Python · FastAPI · PostgreSQL |

Arquitetura consistente em todos os projetos: Supabase como backend, RLS via `is_admin()`, e agentes de IA por módulo quando aplicável.
