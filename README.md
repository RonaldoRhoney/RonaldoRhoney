## 🚀 Sobre Mim

Sou desenvolvedor de software e fundador da **RhoneyInc**, um holding studio de produtos digitais baseado em Belém (PA), Brasil, construído sob o conceito **"Uma conta. Todos os softwares."** — uma plataforma unificada de login para um portfólio de SaaS voltados a B2C e B2B.
